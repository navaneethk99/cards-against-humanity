# Cards Against Humanity – CLI Edition

[![Release](https://img.shields.io/github/v/release/navaneethk99/cards-against-humanity?include_prereleases)](https://github.com/navaneethk99/cards-against-humanity/releases)

A small, terminal-based, fan-made implementation inspired by the Cards Against Humanity party card game. It runs locally in a single terminal and uses a tiny, built-in sample deck for demo play. It is **not** affiliated with or endorsed by Cards Against Humanity LLC.

---

## Features

- Rich-styled terminal UI
- Card Czar selection and judging
- Scoreboard tracking
- Local multiplayer in one terminal
- Online multiplayer with room codes (requires server)
- Installable `clicards` CLI command

## Install & Play

Download the latest release for your operating system and run the client.

macOS:
```bash
./clicards
```

Windows:
```powershell
.\clicards.exe
```

Linux:
```bash
./clicards
```

---

## Play Online

1. Choose `online`.
2. Create a room to get a code, or join using a friend’s code.

---

## Play Local

1. Choose `local`.
2. Enter player names separated by commas.

---

## How To Play

1. A random player becomes the Card Czar.
2. Non-Czar players choose a card from their hand.
3. The Czar picks the winning card; the winner gains a point.
4. Keep playing as long as you want.
